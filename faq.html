Frequently Asked Questions

What is EnginueCore?
EnginueCore is a full Alpine Linux terminal that runs 100% in your browser — with real root access, Docker support, persistent storage, and your own subdomain. No installation required. Works perfectly on desktop, phone, and tablet.
Who created EnginueCore?
EnginueCore was created by Romeo Asis Jr., a full-stack web developer from the Philippines with 12 years of professional experience in the web development industry. He built this project from the ground up with a passion for secure, accessible, and powerful browser-based computing.
Is my data safe and private?
Yes. The free tier runs entirely in your browser — nothing is sent to our servers. Pro plans only store encrypted session data and your email. We do not log commands, files, IP addresses, or terminal activity.

Security features include:
  • End-to-end encryption for all data transfers
  • Automatic session timeout after 30 minutes of inactivity
  • No persistent storage of terminal activity
  • Compliance with GDPR and CCPA regulations
Can I run Docker, nginx, Node.js, or host a real website?
Yes — on Pro and higher plans. You get a real Alpine Linux container with full Docker, Portainer, nginx, Node.js, Python, databases, and more — all behind your own subdomain (e.g., app.enginuecore.com).

Supported technologies:
  • Docker & Docker Compose
  • Node.js, Python, PHP, Ruby
  • Nginx, Apache
  • MySQL, PostgreSQL, MongoDB
  • Git, SSH, and all standard Linux tools
Does it work on mobile?
Absolutely. Full keyboard input, touch support, fullscreen mode, and responsive design. Works flawlessly on iPhone, iPad, Android phones & tablets.

Mobile features:
  • Touch-optimized terminal interface
  • Virtual keyboard with special character shortcuts
  • Fullscreen mode for better focus
  • Gesture controls for common actions
Is this safe for penetration testing or security research?
Yes — and widely used by ethical hackers and red teamers. Isolated environment, no logs, disappears when you close the tab. Use responsibly and legally.

Security research features:
  • Completely isolated environment
  • No IP logging or activity tracking
  • Pre-installed security tools (on Pro plans)
  • Ability to reset environment instantly
What are the pricing plans?
Free Tier: Basic terminal access with limited resources and session duration.

Pro Plan: Full Docker support, custom subdomain, persistent storage, and extended session limits.

Enterprise Plan: Dedicated resources, priority support, team management features, and custom domains.

All plans include our core security and privacy features with no hidden costs.
How do I get started?
Getting started is simple:
  1. Visit our website and create a free account
  2. Choose your plan (free tier requires no payment)
  3. Access your terminal immediately through your browser
  4. Start coding, testing, or learning in your isolated environment
No downloads, no installation, no configuration required.
